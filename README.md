## 🚀 Projetos em Destaque e Estudos

### 🔹 APS Banco de Dados - Window Garden
* **Descrição:** Desenvolvimento de uma aplicação web com um banco de dados de 100 registros botânicos, projetada para ajudar usuários a escolherem as melhores plantas (flores, folhagens ou hortas) para ambientes internos e residenciais.
* **O que usei:** JavaScript, Laragon, VsCode, Html.
* ➡️ [Ver Repositório:](https://github.com/mewissagatos/WindowGarden.git)

### 🔹 APS Lógica Programação - Quiz
* **Descrição:** Código desenvolvido em terminal (sem *front-end*) para conscientizar o usuário sobre os perigos da internet, utilizando um sistema de pontos para demonstrar o quão bem protegido ele está.
* **O que usei:** Java
* ➡️ [Ver Repositório](https://github.com/alessonalvesg10/APS-Programacao.git)

### 🔹 APS Lógica Matemática - Pensamento de tabela verdade
* **Descrição:** Código desenvolvido utilizando lógica proposicional como base para a tomada de decisões. O sistema monitora a saúde e as condições da planta para, quando necessário, realizar ações automatizadas como ativar a irrigação, recolher o toldo retrátil ou acionar defesas contra pragas.
* **O que usei:** Java.
* ➡️ [Ver Repositório:](https://github.com/CoDynames/Smart-Garden.git)
